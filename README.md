# User Activation and Retention Analysis for a SaaS Platform

## Overview
This project analyzes the user journey from signup to activation and long-term retention for a SaaS platform. The objective is to identify onboarding friction, high-value user segments, and opportunities to improve Day 30 retention.

## Business Problem
Despite strong top-of-funnel user acquisition, leadership observed weak activation and inconsistent retention across user segments. This analysis evaluates where users drop off and which behaviors are associated with long-term engagement.

## Stakeholders
- Product Manager
- Growth Team
- Lifecycle Marketing
- Senior Leadership

## Dataset
Tables used:
- users
- events
- subscriptions
- marketing_spend

## KPI Definitions
- Activation Rate
- Day 7 Retention
- Day 30 Retention
- Trial-to-Paid Conversion
- Feature Adoption Rate

## Methodology
- Built SQL analysis for funnel, retention, and segmentation
- Measured onboarding conversion through key milestones
- Created cohort-based retention views
- Compared segment performance across channels, devices, and plan types
- Identified early behaviors linked to retention

## Key Findings
- Largest drop-off occurred between signup and first meaningful product action
- Referral and organic users had the strongest Day 30 retention
- Users who created their first report within 3 days retained at a much higher rate
- Trial users in mid-sized companies showed the strongest paid conversion

## Recommendations
1. Reduce onboarding friction before first report creation
2. Trigger lifecycle nudges for users inactive within first 3 days
3. Rebalance acquisition spend toward higher-retention channels

## Business Impact
This analysis highlights opportunities to improve activation efficiency, increase retained users, and improve downstream paid conversion.
